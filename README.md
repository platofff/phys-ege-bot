# Telegram-бот для подготовки к ЕГЭ по физике

### Скачивание задач ФИПИ
```commandline
pip install -r download_tasks_requirements.txt
python download_tasks.py
```

### Запуск чат-бота
```commandline
pip install -r requirements.txt
python main.py
```
